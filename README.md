# 📚 Library System

Um sistema de gerenciamento de biblioteca desenvolvido em Java, 
criado com o objetivo de aprender os fundamentos da linguagem 
e orientação a objetos.

## Funcionalidades

- Listar livros do acervo
- Emprestar livros
- Devolver livros

## Tecnologias

- Java 21
- VS Code

## Como executar

1. Clone o repositório
2. Abra a pasta no VS Code
3. Execute o arquivo `Main.java`

## Aprendizados

- Classes e objetos
- ArrayList
- Métodos
- Scanner (entrada do usuário)
- Separação de responsabilidades
